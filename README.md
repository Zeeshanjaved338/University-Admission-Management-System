# University Admission Management System

## Overview
The **University Admission Management System** is a console-based C# application designed to manage university admissions efficiently. It allows administrators to manage degree programs, subjects, and student information, providing a structured way to handle the admission process.

This project demonstrates object-oriented programming, file handling, and console application development in C#.

---

## Features
- Add, view, and manage degree programs
- Add, view, and manage subjects
- Add and manage student information
- Assign subjects to degree programs
- Simple console-based user interface
- Persistent data storage using files
